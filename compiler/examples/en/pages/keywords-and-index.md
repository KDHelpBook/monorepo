# Keywords and the index

Every page can declare keywords in its frontmatter:

```yaml
keywords: [Timer, interval, scheduling]
```

All keywords across all pages are collected into the alphabetical **Index** tab —
the modern equivalent of pressing **F1** in an editor. If a keyword points at more
than one page, the viewer expands a small chooser.

## Good practice

- Name keywords the way a reader would search for them.
- Add synonyms (for example `scheduling` next to `Timer`).
- A few precise keywords beat a dozen vague ones.

Keywords are also weighted in [full-text search](what-is-khb).
